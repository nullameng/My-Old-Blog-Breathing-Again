我的台東遊蕩法

文章日期：2009-08-06 07:55

上一回去台東悠閒地玩個幾天，已經是兩年半以前的事情了。剛剛搜尋舊資料夾，發現一個小筆記，是當時在某個網路論壇上，回答網友的問題：既然「回台東懷舊，能否談談您覺得最有意思的地方？」

我的回覆是：

...
談到這個嘛，我就聊一聊這幾年去過的點

台東海岸
天氣：
台東夏天實在熱，海岸地區正午以前烈日毒辣曝曬。午後有雲，比較宜人。
冬天則大抵比台灣其他地方溫暖（恆春除外）。

旅遊點（由南向北）：
# 整個台11線沿途有多處方便停車的觀景點和遊憩區，不一一介紹。
# 小野柳：海岸奇石。
# 杉原海水浴場：當然是夏天的好去處。不過現在正大興土木蓋休閒飯店。
# 杉原以北的海岸有些餐飲店，我去過黃金海岸，景觀好（還有名喚「大頭」的拉布拉多獵犬一隻）。
左巴花園就在海水浴場邊，這次經過好像已經消失了？
# 水往上流：看起來完全沒有往上流的感覺或錯覺，讓我這個「老台東」覺得臉紅。就算路過也不值得去。
# 東河包子：確實可口，可謂名不虛傳，適合海岸旅遊中途補給充飢。
# 成功國立海洋生物展覽館：經營尚稱用心，若時間允許，不妨一遊；小朋友會喜歡海洋生物觸摸池。（......這裡2008年底結束營業，重新招標，我不知道重新開幕了沒有，要去以前得打聽一番。）
# 三仙台：巨大奇岩的離岸島。應該是不可不去的點。
在島上或橋上回頭眺望向北向南延伸的海岸線以及臨海山脈，壯觀。站在橋上向下看海浪，由於海浪繞射的關係，浪會從兩邊在橋下（腳下）相撞，浪大時頗具震撼力。
**警告：夏天奇熱，要帶水。** 實在怕熱就下午三點以後去。
# 八仙洞：從台東出發...好遠！小學時去過一次，當時完全不覺得有何好玩之處，所以...長大後從未再去。

典型的夏天行程：
在飯店吃個慵懶豐盛的早餐後出發向北，目標：成功及三仙台。
午餐可在成功吃海鮮，或者，如果早餐非常非常豐盛，可以跳過午餐，回程在東河吃包子。
1500以後（太陽比較不毒）到杉原海水浴場，或者小野柳，傍晚在富崗吃海鮮，或者回台東市用餐。

縱谷
天氣：
夏天正午以前有雲的機會比海岸地帶要多些；午後雲更多，也可能有陣雨。
冬天風不小。

# 卑南山/四格山頂：標高約400公尺，展望良好，開車可達，步行5分鐘即可到山頂。
我喜歡高處有展望的地方，可以由不同的角度看熟悉的台東。沒興趣或者對台東不熟的朋友，恐怕會覺得無聊。
# 卑南文化公園：有閒不妨去逛逛。
# 初鹿牧場：應該是可去可不去。有草有牛，牧場嘛。
我七年前去了一趟，回家後寫了這樣的感想：
「小小的牧場中不大看得到成群結隊的乳牛徜徉在一望無際的草原上...我去的時候只看到所有的乳牛關在小我牛欄中，每一頭牛都是渾身污泥（和....排泄物？），根本就是倒足胃口；如果給小朋友看到了，妳們猜猜看他以後願不願意再喝牛奶！也許當時只是特殊狀況（有牛偷了牧場主人的錢，東窗事發後沒有牛承認是自己幹的好事，所以所有牛被關起來拷問）。希望你們的運氣較好，可以看到土地銀行（牧場的經營者）把牛洗的乾乾淨淨後放到草地上載歌載舞。」
前年再去，倒是沒看到過於倒胃口的鏡頭。小朋友餵餵牛、推推「草包」、吃吃鮮乳冰淇淋，也可以消磨些時間吧。
# 應用原生植物園：我一年半以前去過一次。其實不算景點，是個台東有機生技農業產品的展售中心，進場要門票，可以抵消費。我對這類產品毫不熱中，不過...此處建築特殊，還有個延伸近百公尺的觀景平台，展望相當好（可惜當時平台末端有強烈豬圈臭味，無法真的悠閒觀景！），如果很閒，可以在該處消磨個把小時，順便買些禮品吧。
# 鹿野高台：有幾處瞭望台，展望佳。
# 關山：環保親水公園，設計用心，景觀融入縱谷兩側群山，建議一定要去。可於公園外租自行車後騎入；自行車道一圈長度超過兩公里。
園區景緻難以筆墨形容，請參觀網站：
http://www.wintimes.com.tw/play/950/956/956-01-04.htm
關山還有環鎮自行車道：因為孩子還小，我其實沒騎過環鎮車道，但是心嚮往之。
# 池上：小時候遠足去池上是去大坡池（然後吃池上飯包）；現在好像還是只有大坡池...和池上飯包。我往往把池上當作在關山騎自行車後吃午餐的地方；任選一家號稱池上飯包的店家，口味都不離譜（重點是：米飯好吃）。

典型的夏天行程：
在飯店吃個慵懶豐盛的早餐後出發向北，目標：關山。途中經過鹿野高台看風景。關山騎車1-2小時，午餐可在池上或關山吃。回程逛逛初鹿牧場。
若下午縱谷下起雨來，我會經卑南轉「台11乙」開向海岸較近的景點，或者改為室內行程，如史前文化博物館。

台東市：
# 森林公園（基本上就是台東市的防風林），可騎自行車，但是秋天冬天東北風大的日子，有風沙的問題，並不宜人！
我住台東的時候，風沙極為嚴重，一旦起了風沙，我從家中無法看清800公尺外的鯉魚山；施孝榮唱「歸人沙城」，我認定他指的就是台東。現在建立了防風林，台東市的風沙若和童年印象相比，已經大幅改善。不過真的跑進防風林中...小時候的記憶可就跑回來囉！
# 台東米苔目/臭豆腐等等小吃：許多書籍均有介紹，請自行嘗試。卑南豬血湯到底為什麼有名，我嘗試後...還是弄不明白！
# 馬蘭台東糖廠販賣部：在地人吃冰的地方。這個嘛，嘿嘿，書上可就不太找得到囉！
# 國立史前文化博物館：市郊民航機場附近。國內最精緻的博物館之一，值得參觀。目前（2007年春）暴龍「蘇」特展恰巧巡迴到史前文化博物館，展期到5月27日。

知本：就是溫泉和溫泉旅店囉。
# 知本老爺服務及餐飲俱佳，當然價位是高了些；如果連住三夜以上，有「新移民台東」的優待方案，個人認為算是物超所值。
我沒住過知本的其他溫泉旅店，給不出建議。
住知本遊台東有個小問題：我喜歡的遊玩點全在台東市以北，如果用知本當作基地每天向北跑，光是來回車程每天就得（比住在台東市）多花上四五十分鐘。
# 知本森林遊樂區：長大後沒去過，無法給建議。

太麻里以南...我小學入學是在金峰鄉的歷坵國小！長大後則從沒去玩過。
像金針山這樣的熱門景點，我就無法給建議了。
