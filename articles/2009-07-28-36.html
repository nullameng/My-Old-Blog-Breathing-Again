在下這就為各位隆重介紹那仙樂，「書信二重唱」sull'aria
出自費加洛婚禮，第三幕。

不是說，Susanna和伯爵夫人用Susanna的名義寫了封信給伯爵，說是答應和他晚上在庭院裡幽會嗎？
這曲子就是寫那「開會通知」的過程，夫人口述，Susanna抄寫。

http://tw.youtube.com/watch?v=CqfifgIASkI
那個「啟蒙」版本，音效有些問題。
http://tw.youtube.com/watch?v=PmlCpOojau8
這是Gardiner的版本，使用古樂器，比現代的樂器低了半個音。
http://tw.youtube.com/watch?v=gaVIwwNhocg
噫，最後一個音怎麼被切掉了呢？
http://tw.youtube.com/watch?v=wd5nFd3utLg
Popp很棒，伯爵夫人唱的不錯...演技不佳。

歌詞如下。

LA CONTESSA: / ROSINA:

"Canzonetta sull'aria ... " / "A little song on the breeze ... "

SUSANNA:
[scrivendo] / [writing]
"Sull'aria ... " / "On the breeze ... "

LA CONTESSA: / ROSINA:
"Che soave zeffiretto ... " / "What a gentle zephyr ... "

SUSANNA:
"Zeffiretto ... " / "Zephyr ... "

LA CONTESSA: / ROSINA:
"Questa sera spirera ... " / "Will whisper this evening ... "

SUSANNA:
"Questa sera spirera ... " / "Will whisper this evening ... "

LA CONTESSA: / ROSINA:
"Sotto i pini del boschetto." / "Beneath the pines in the copse."

SUSANNA:
"Sotto i pini ... " / "Beneath the pines ..."?

LA CONTESSA: / ROSINA:
"Sotto i pini del boschetto." / "Beneath the pines in the copse."

SUSANNA:
"Sotto i pini ... del boschetto ... " /"Beneath the pines ... in the copse ... "

LA CONTESSA: / ROSINA:
Ei gia il resto capira. / And he'll understand the rest.

SUSANNA:
Certo, certo il capira. / Oh, yes, certainly he'll understand it.
